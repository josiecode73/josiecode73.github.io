關於我 (About)

用設計解決問題，用程式實現想像。

你好，我是 Josie，一名專注於網站設計 (Web Design) 與前端開發 (Frontend Development) 的設計工程師。

我深信一個優秀的網站不僅需要出色的視覺美感，更需要流暢的使用者體驗與穩健的程式架構。在過去的專案經驗中，我致力於協助企業與品牌打造具備現代感、高轉換率且響應式的網頁。

從前期的需求訪談、Wireframe 規劃、UI/UX 設計，到後期的 HTML/CSS/JS 前端切版與動態效果實作，我能獨立完成網站的從無到有，確保設計理念被 100% 完美還原。
設計師 Josie
核心理念 極簡主義 (Minimalism) / 使用者為中心 (User-Centered Design) / 像素級還原 (Pixel-Perfect)
設計工具 Figma / Adobe XD / Photoshop / Illustrator / Adobe XD
開發技能 HTML5 / CSS3 / JavaScript (ES6+) / React / Tailwind CSS / RWD 響應式開發
服務項目
  • 企業形象官網設計：打造具備品牌識別度的專業企業網站。
  • 品牌電商介面設計：優化購物流程，提升使用者體驗與轉換率。
  • 前端網頁開發：將設計圖轉換為流暢且兼容各裝置的動態網頁。

  • 網站維護與優化：現有網站的效能調校、視覺翻新與維護。
